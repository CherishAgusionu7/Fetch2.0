# Fetch: The Clean Water Quest

Fetch is a browser-based platformer game built with React, TypeScript, Vite, and Canvas rendering. The player explores a colorful world, collects clean water, and delivers it to families in need while avoiding hazards and pollution creatures.

## Features

- Responsive 2D platformer gameplay
- Keyboard and mobile touch controls
- Water collection and delivery missions
- Hazard-filled level design with enemies and obstacles
- HUD, mission prompts, and end-of-game screens
- Built-in audio and particle effects

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Canvas-based game rendering
- Playwright for smoke testing

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the game in your browser at:
   ```text
   http://localhost:3000/
   ```

## Available Scripts

- `npm run dev` — start the local development server
- `npm run build` — create a production build
- `npm run preview` — preview the production build locally
- `npm run lint` — run TypeScript checks
- `npm run test:smoke` — run a basic browser smoke test

## Controls

- `A` / `D` or `←` / `→` — move
- `W`, `↑`, or `Space` — jump
- `E` — refill or deliver water

## Notes

The game is designed to be played in a modern browser. For the best experience, use a desktop browser or a mobile device with touch support.
